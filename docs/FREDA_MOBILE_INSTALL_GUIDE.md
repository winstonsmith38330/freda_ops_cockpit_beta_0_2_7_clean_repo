# Freda Mobile Install Guide — Beta 0.2

## Best setup for next week

Beta 0.2 should be deployed as a Node app over HTTPS if Freda needs live sales.

A static deployment will show the interface and sample/offline data, but the **Sync reporting.site POS** button needs the server.

## iPhone install

1. Open the HTTPS app link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Name it `Freda Ops`.
5. Open from the home screen.

## Android install

1. Open the HTTPS app link in Chrome.
2. Tap Install app or Add to Home screen.
3. Open from the home screen.

## Daily use

1. Open Freda Ops.
2. Tap Refresh live summary.
3. If needed, tap Sync reporting.site POS.
4. Add Uber/Square snapshot if not yet captured.
5. Ask: “What needs my attention today?”

## What Freda should expect

The app should give direct actions, not long reports:

- Which store needs attention.
- Whether POS/Uber/Square has been captured.
- What manager follow-up is needed.
- What production/store-standard risk exists.
